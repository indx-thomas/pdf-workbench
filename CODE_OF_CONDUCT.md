# Code of conduct

Contributors and participants are expected to be respectful, constructive, and considerate.

Harassment, personal attacks, discrimination, intimidation, publication of private information, and intentionally disruptive behaviour are not acceptable. Technical disagreement is welcome when it remains focused on evidence and the work.

Project maintainers may edit, remove, or reject contributions and participation that violate these expectations. Conduct concerns can be raised privately with the repository owner, `@indx-thomas`.
